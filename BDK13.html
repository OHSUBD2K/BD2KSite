Learn FHIR

Description

Tutorial for implementing the SMART on FHIR standards for clinical data.

Objectives

At the completion of this component, the learner will be able to:

  • Express clinical data using the Fast Health Interoperability Resources (FHIR) standard
  • Implement applications in the Substitutable Medical Apps, Reusable Technology (SMART) framework
